# Screenshots

Esta pasta é destinada a armazenar as capturas de tela (screenshots) da aplicação.
Elas serão referenciadas no `README.md` principal na raiz do projeto.

## Arquivos esperados

Certifique-se de adicionar e nomear os seguintes arquivos para que apareçam corretamente na documentação:

- `dashboard.png` (Tela inicial do painel financeiro)
- `login.png` (Tela de autenticação)
- `financeiro.png` (Tela de controle de contas/invoices)

Se precisar adicionar novas telas, atualize tanto esta pasta quanto a seção respectiva no `README.md` raiz.
