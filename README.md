# Alisson Joias ERP

Projeto do teste tecnico full stack para um modulo financeiro de ERP de joalheria, inspirado no contexto PIRO Fusion.

## Setup inicial

O scaffold base foi criado com `create-next-app` usando:

- Next.js App Router
- TypeScript
- Tailwind CSS
- ESLint
- npm

O projeto Supabase local foi inicializado com Supabase CLI em `supabase/config.toml`. As bibliotecas oficiais instaladas para a futura integracao Next.js + Supabase sao `@supabase/supabase-js` e `@supabase/ssr`.

## Rodando localmente

```bash
npm install
npm run dev
```

A aplicacao fica disponivel em `http://localhost:3000`.

## Documentacao

Antes de desenvolver qualquer feature, leia `AGENTS.md` e a documentacao em `docs/`.

Os PDFs originais do teste e das pesquisas nao devem ser versionados neste repositorio.
